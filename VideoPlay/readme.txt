[ VideoPlay 사용 설명서 ]


 1. 어플 내용

 동영상 화면을 캡쳐하여 원하는 Object를  crop하여 저장할 수 있는 기능을 지원합니다.




 2. 사용방법

  
1) 어플을 실행시키면 비디오 동영상 파일이 생성및 실행됩니다. 

   해당 동영상에서 캡쳐 하고자 하는 영상이 나타났을 때, Capture Video 버튼을 눌러 화면을 정지시키면,

   정지된 해당 영상의 프레임이 캡쳐가 되어 동영상 하단에 나타납니다.


2) 프레임에서 원하는 이미지를 드래그하여 crop 합니다.


3) Crop&Save 버튼을 클릭하여  crop한 이미지를 저장합니다.


4) 저장된 파일은 하단에 이미지로 나타나기도 하지만, 가상 Local storage에서도 저장 되었음을 확인할 수 있습니다.

   ( Local storage 는  F12를 누른 후 확인 가능합니다.)
